This repository contains various Go language labs and exercises for learning and practicing Go programming. Each lab focuses on different aspects of the Go language, including basic syntax, data structures, concurrency, and more.

## Lab Structure

Each lab is organized in a separate directory with a descriptive name. Inside each directory, you will find Go source files (.go) that contain the code for the exercises. You can run these files using the Go command-line tool.

## Getting Started

To get started with the labs, follow these steps:
1. Clone the repository to your local machine:
```bash
git clone https://github.com/chethanyadav456/go_lab_DCSCS.git
```
2. Navigate to the desired lab directory:
```bash
cd go_lab_DCSCS/01_hello
```
3. Run the Go source file:
```bash
go run hello.go
```
